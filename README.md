# Bread-Client
The Newest And Tastiest Hack Client!

## Features
- Custom HUD with configurable color
- Configurable hotkey (default H) to toggle HUD
- Fly mode toggle in config

## Building
1. Ensure you have Java 21 installed.
2. Run `./gradlew build` to build the mod.
3. The mod jar will be in `build/libs/`.

## Installation
Place the jar in your `.minecraft/mods` folder.

## Usage
- Press H to toggle the HUD.
- Configure settings via Mod Menu (if installed) or config file.
